## Простые выражения

```python
>>> 5 + 2     # => 7
>>> 5 / 2     # => 2
>>> 5 // 2    # => 2
>>> 37 / 3.0  # 12.333333333333334
>>> 37 // 3.0 # 12.0
```



## Вывод на экран

```python
>>> print("Hello, World!")
Hello, World!
```



## Пользовательский ввод

```python
>>> hello = input()
"Hello!"
>>> print(hello)
Hello!
```
